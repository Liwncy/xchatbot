# 修仙插件（xiuxian-plugin）

> 基于微信文本消息的修仙玩法插件。所有指令统一以 `修仙` 开头，支持创建角色、修炼、探索、装备、经济、拍卖、BOSS 讨伐、爬塔赛季、灵宠、情缘等模块。

- 插件入口：`src/plugins/game/xiuxian/index.ts`
- 触发条件：文本消息以「修仙」开头
- 数据依赖：Cloudflare D1（`XBOT_DB`）+ KV（`XBOT_KV`）
- 未识别指令会提示发送「修仙帮助」查看完整菜单

## 快速上手

```text
修仙创建 [名字]       # 创建角色（不填名字则使用昵称）
修仙状态              # 查看个人面板
修仙修炼 [次数]       # 修炼获取经验
修仙探索              # 随机探索事件
修仙背包 [页码] [筛选] # 查看背包，支持品质/类型/排序筛选
修仙帮助 [分类]       # 查看指令菜单
```

发送 `修仙帮助` 可查看插件内置的完整指令列表；发送 `修仙帮助 <分类>` 可查看特定分类。

## 指令菜单

以下列表与插件内 `修仙帮助` 输出保持一致。方括号 `[]` 内为可选参数，`|` 表示多选一。

### 基础

| 指令 | 说明 |
|---|---|
| `修仙创建 [名字]` | 创建修仙角色 |
| `修仙状态` | 查看个人面板（境界 / 气血 / 战力 / 装备 等） |
| `修仙修炼 [次数]` | 修炼获取经验 |
| `修仙探索` | 触发随机探索事件 |
| `修仙背包 [页码] [类型/品质/排序]` | 查看背包，支持筛选与排序 |
| `修仙装备 [编号]` | 装备背包中的道具 |
| `修仙卸装 [武器|护甲|灵宝|法器]` | 卸下某槽位装备 |

品质关键词：`普通(白) / 优秀(绿) / 稀有(蓝) / 史诗(紫) / 传说(金) / 神话(红)`。

背包示例：`修仙背包 1 神话 评分降序`。

### 经济

| 指令 | 说明 |
|---|---|
| `修仙商店` | 查看今日商店 |
| `修仙购买 [商品ID]` | 从商店购买 |
| `修仙出售 [装备ID...]` / `修仙出售 全部` / `修仙出售 品质 稀有以上|稀有以下` | 出售装备 |
| `修仙分解 [装备ID...]` / `修仙分解 全部` / `修仙分解 品质 稀有以下` | 分解装备 |
| `修仙上锁 [装备ID...]` / `修仙解锁 [装备ID...]` | 防止误售 / 误分解 |
| `修仙炼器 [装备ID] [次数|无限]` | 炼器强化 |
| `修仙炼器详情 [装备ID]` / `修仙炼详 [装备ID]` | 查看炼器详情 |
| `修仙流水 [条数]` | 查看经济流水 |
| `修仙上架 [装备ID] [起拍价] [时长分钟] [一口价]` | 发布拍卖 |
| `修仙拍卖 [页码]` | 查看拍卖列表 |
| `修仙竞拍 [拍卖ID] [出价]` | 参与竞拍 |
| `修仙秒拍 [拍卖ID]` | 以一口价成交 |
| `修仙撤拍 [拍卖ID]` | 撤销自己的拍卖 |
| `修仙拍结 [拍卖ID]` | 结算已到期拍卖 |

### 成长

| 指令 | 说明 |
|---|---|
| `修仙签到` | 每日签到 |
| `修仙任务 [可领]` | 查看任务；`可领` 只展示可领奖的任务 |
| `修仙领奖 [任务ID]` / `修仙领奖 全部` | 领取任务奖励 |
| `修仙成就` | 查看成就 |
| `修仙占卜` | 每日占卜抽取当日运势（作用于修炼/探索/战斗） |
| `修仙运势` | 查看今日卦象与 buff |
| `修仙改运` | 消耗灵石重抽今日卦象（成本递增，每日最多 4 次） |
| `修仙奇遇` | 触发 NPC 奇遇 |
| `修仙奇录 [页码]` | 查看奇遇记录 |
| `修仙结缘 [@对方 / 对方wxid]` | 发起结缘 |
| `修仙允缘` / `修仙拒缘` | 同意 / 拒绝结缘 |
| `修仙解缘` | 解除情缘 |
| `修仙同游` | 情缘同游 |
| `修仙情缘` | 查看情缘状态 |
| `修仙情录 [页码]` | 查看情缘记录 |

### 斗法（PvP）

| 指令 | 说明 |
|---|---|
| `修仙切磋 [@对方 / 对方wxid]` | 发起切磋 |
| `修仙应战` / `修仙拒战` | 响应切磋邀请 |
| `修仙强斗 [@对方 / 对方wxid]` | 强制开战（也可用 `修仙强战` / `修仙强制战斗`） |

> 当前强斗规则：发起冷却 30 秒、目标保护 30 秒；胜利后可掠夺对方灵石，基础比例 8%、基础上限 120，并会按双方等级差动态调整——越级强斗收益更高，压级强斗收益更低。

| 指令 | 说明 |
|---|---|
| `修仙战报 [页码]` | 查看个人战报列表（含切磋 / 强斗 / 挑战 等） |
| `修仙战详 [战报ID]` | 查看战报详情 |

### 帮助

| 指令 | 说明 |
|---|---|
| `修仙帮助 [分类]` | 查看菜单；不带分类则查看总览 |
| `修仙指令` / `修仙菜单` | 与 `修仙帮助` 等价 |

支持的帮助分类：`基础 / 经济 / 成长 / 斗法 / 讨伐 / 爬塔 / 灵宠 / 战报 / 全部`；
也支持常用别名（示例）：`商店→经济`、`切磋/强斗→斗法`、`赛季/塔榜→爬塔`、`宠物→灵宠`。

## 目录结构

```text
src/plugins/game/xiuxian/
  index.ts          # 插件入口（匹配「修仙」前缀 + 分发）
  app/commands/     # 指令解析（文本 → XiuxianCommand）
  app/reply/        # 顶层与通用回复文案
  app/service.ts    # 指令分发与业务主逻辑
  features/battle/  # 挑战战报、战详文案
  features/boss/    # 讨伐、世界BOSS、BOSS战报文案
  features/economy/ # 商店、拍卖、出售、分解、炼器、流水文案
  features/fortune/ # 占卜运势 feature 与 buff 计算
  features/growth/  # 签到、任务、领奖、成就文案
  features/pet/     # 灵宠面板、宠包、喂宠、出战切换文案
  features/social/  # 奇遇、结缘、切磋/强斗文案
  features/tower/   # 爬塔、赛季、塔战报文案
  core/balance/     # 数值、掉落、套装与战斗计算
  core/constants/   # 全局玩法常量与文案术语
  core/repository/  # D1 / KV 数据访问
  core/types/       # 领域类型与命令协议
  core/utils/       # 时间、境界等通用工具
```
